# CNN Workflow – Homework 1

## Overview
This project implements a Convolutional Neural Network (CNN) workflow for image classification as part of Homework 1.

The notebook demonstrates the full deep learning pipeline from loading the dataset through preprocessing, model building, training, evaluation, and performance analysis.

---

## Objectives
- Build and train a CNN model for image classification
- Apply preprocessing and normalization to image data
- Evaluate model performance using validation/testing
- Understand the CNN workflow in practice

---

## Workflow
The notebook follows these steps:

1. Import required libraries
2. Load and inspect the dataset
3. Preprocess and normalize images
4. Split data into training and testing sets
5. Build the CNN architecture
6. Train the model
7. Evaluate performance
8. Visualize results

---

## Tools and Libraries
- Python
- Jupyter Notebook
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

---

## Model Components
The CNN includes:
- Convolutional layers
- Activation functions (ReLU)
- Pooling layers
- Flatten layer
- Dense / fully connected layers
- Output layer for classification

---

## Results
The model was trained and evaluated on the dataset.

Performance was reviewed using:
- Training accuracy
- Validation accuracy
- Loss curves
- Final predictions

---

## Key Learning Points
- CNNs automatically learn image features
- Preprocessing improves training consistency
- Validation helps monitor overfitting
- Visualizing metrics helps understand performance
